AETHER GAZER SEMI-AUTO

深空之眼 Sid 半自動戰鬥輔助系統

□ View Project on GitHub
CORE SYSTEM

自動戰鬥核心

  • • 自動判定技能 CD 與可用狀態
  • • 像素級圖片偵測 判斷機制
  • • WASD 移動優先權，不干擾手操
CHARACTERS

角色特化模式

內建多款主流角色專屬偵測，針對特定技能圖示強化判斷：
魂羽 / 緋染 / 巧构 / 庚辰

※ 其他角色功能開發中
OVERLAY

狀態疊圖系統

即時顯示 腳本狀態 / 戰鬥動作。優化按鍵發送機制，直接傳送指令至視窗，大幅降低輸入衝突。

重要

必須從遊戲設定開啟「鍵位提示」

腳本以鍵位提示中的 ESC 按鈕 作為戰鬥狀態判斷依據。

SID AUTOMATION LAB • 數據環境校準完成

技術核心：影像辨識

基於螢幕畫面模擬技術，完全不讀取記憶體、不攔截遊戲封包。這不是傳統外掛，而是協助你更輕鬆遊戲的數位助手。

非侵入式架構
開發者堅持

我拒絕自動瞄準與全自動掛機。我只鎖定冷卻時間、協助壓槍與自動補給。把無意義的勞動交給程式，把樂趣留給你。
